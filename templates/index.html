Parâmetros Disponíveis

Parâmetro Descrição Padrão Exemplo
limit Número máximo de artigos 30 ?limit=20
pages Páginas para varrer 3 ?pages=5
q Filtrar por termo no título/descrição - ?q=Flamengo
source_url URL de origem alternativa /mais-noticias ?source_url=...
refresh Forçar nova varredura 0 ?refresh=1

Exemplos de Uso

RSS com 20 artigos
{{ request.url_root }}feeds/lance/rss.xml?limit=20
Atom filtrado por "Flamengo"
{{ request.url_root }}feeds/lance/atom.xml?q=Flamengo
RSS com 5 páginas
{{ request.url_root }}feeds/lance/rss.xml?pages=5
Atom com atualização forçada
{{ request.url_root }}feeds/lance/atom.xml?refresh=1
